TOPIC: "Lot Attribute Setup" — Step 3: Auto-Population Fields

WHAT THIS STEP COVERS:
Auto-population fields eliminate manual data entry by capturing lot metadata automatically during receiving. PO Lot Attribute, PO Line No. Lot Attribute, Vendor Parse Attribute, Vendor Code Parse Attribute, and 3PL Lot Attribute (e.g., FREEZERLOT) each map to a lot attribute that the system fills in without user intervention.

DETAILED CONTEXT:
Setting "3PL Lot Attribute" to FREEZERLOT means every time you receive, the system auto-captures the cold storage lot number. When your 3PL calls about lot F12345, you can instantly find it. PO and Vendor parse attributes auto-stamp the PO number and vendor code onto every lot for instant cross-reference.

KEY NARRATION POINTS:
Auto-population fields are where the Lot Attribute Setup saves your receiving team significant time and eliminates data entry errors. Five fields on the setup page map specific lot attributes to automatic capture during the receiving process. The PO Lot Attribute stamps the Purchase Order number onto every lot created from that PO — so you can always trace a lot back to the order that brought it in. Lot Attribute does the same for the line number, giving you line-level traceability. Vendor Parse Attribute and Vendor Code Parse Attribute automatically capture the vendor name and vendor code onto the lot, which is invaluable when you need to trace a quality issue back to the supplier.
