TOPIC: "Lot Attribute Setup" — Step 1: Global Attributes (1-8)

WHAT THIS STEP COVERS:
Global Attributes 1 through 8 appear as COLUMNS on the Item Lot Attributes page. Each global attribute maps to a lot attribute code (e.g., MYGRADE, COO) and defines one axis of the item-level attribute matrix. These are the primary dimensions that distinguish one lot configuration from another.

DETAILED CONTEXT:
When you set COO as Global Attribute 2, every item in the system can track country of origin per lot. This is critical for FDA compliance, tariff calculations, and customer-specific sourcing requirements. Up to 8 global attributes can be active simultaneously.

KEY NARRATION POINTS:
Global Attributes are the structural columns of your lot tracking system. The Lot Attribute Setup page provides eight global attribute slots, numbered 1 through 8, and each one maps to an attribute code you have defined. For example, you might set Global Attribute 1 to MYGRADE — your quality grading system — and Global Attribute 2 to COO for country of origin. Once assigned, these attributes appear as columns on the Item Lot Attributes page for every item in the system. This is what creates the matrix.
