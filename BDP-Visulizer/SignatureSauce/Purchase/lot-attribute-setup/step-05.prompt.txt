TOPIC: "Lot Attribute Setup" — Step 5: Item Lot Attributes Matrix

WHAT THIS STEP COVERS:
The Item Lot Attributes page shows the matrix created by Global Attributes for each item. Each row is a unique combination of global attribute values. For example, item IM1029 might have 4 rows: MyG_A x CN, MyG_A x US, MyG_B x CN, MyG_B x US — each representing a valid lot configuration.

DETAILED CONTEXT:
Grade A salmon from Norway costs $12/kg while Grade B from Chile costs $8/kg. The matrix lets you set different costs, vendors, preferred UOM sizes, and reorder policies per combination. This is where abstract setup becomes concrete business rules.

KEY NARRATION POINTS:
The Item Lot Attributes page is where all the setup comes together into a concrete, item-specific matrix. Open any item card, navigate to the Item Lot Attributes action, and you see a grid where each row represents a unique combination of your global attribute values. Take item IM1029, Atlantic Salmon Fillets. If MYGRADE has values A and B, and COO has values CN and US, you get four rows: Grade A from China, Grade A from the US, Grade B from China, and Grade B from the US. Each row can carry its own unit cost, preferred vendor, case size configuration, and reorder parameters.
