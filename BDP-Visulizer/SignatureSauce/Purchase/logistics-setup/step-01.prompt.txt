TOPIC: "Logistics Setup" — Step 1: Order Types

WHAT THIS STEP COVERS:
Each Logistics Order has a Type that determines its validation rules and available actions: Ocean, Dray Ocean, Rail, Truck, and Dray. Selecting the correct type unlocks the right fields, workflows, and container management features for that transport leg.

DETAILED CONTEXT:
Ocean orders support full container management — how many 20ft or 40ft containers, seal numbers, temperature logs. Truck orders focus on pallet counts and weight limits. Dray handles the short haul from port to warehouse.

KEY NARRATION POINTS:
Every Logistics Order starts with a Type, and the type you select determines everything about how the order behaves. There are five types: Ocean, Dray Ocean, Rail, Truck, and Dray. Each one unlocks different validation rules and functionality tuned to that transport mode. Ocean orders, for example, enable full container management — you can define how many 20-foot and 40-foot containers are on the shipment, assign seal numbers, and attach temperature logs for reefer containers. This is exactly what you need for that salmon coming across the Atlantic in a temperature-controlled container.
