TOPIC: "Alt Uom Setup" — Step 4: Variable Weight & Quantity Controls

WHAT THIS STEP COVERS:
Default as Variable Weight sets new items to variable weight by default. Allow Quantity Edit on CASE Items controls whether users can change quantities on CASE-tracked lines. CASE Weight Rounding Method determines how fractional weights are handled.

DETAILED CONTEXT:
Seafood is classic variable weight — each box of salmon weighs differently. With Variable Weight ON, the receiver enters actual weight per case instead of relying on a fixed conversion. The rounding method matters for invoicing accuracy on high-volume items.

KEY NARRATION POINTS:
Not every product comes in a perfectly consistent weight. A case of frozen salmon fillets might weigh 10.2 KG one day and 9.8 KG the next. That is where variable weight comes in. The Default as Variable Weight toggle sets the default for newly created items — in a seafood or fresh meat operation, you would turn this ON so every new item assumes variable weight unless you override it. Allow Quantity Edit on CASE Items determines whether users can adjust quantities on lines that involve CASE-tracked items — important for receiving adjustments.
