TOPIC: "Alt Uom Setup" — Step 6: Item Card Configuration

WHAT THIS STEP COVERS:
Per-item fields on the Item Card: CASE Size, Enable CASE Ledger, Input in Alt UOM or Base UOM, Inbound Weight Method (Specific / Standard / Calculated), and Always Ship Full CASE. These override global defaults for individual items.

DETAILED CONTEXT:
Flour uses Standard weight — every box is exactly 10 KG, no weighing needed. Fresh fish uses Specific weight — the receiver weighs each box individually. Calculated weight uses the CASE Size conversion factor. Always Ship Full CASE prevents partial-case picks for items where breaking a case destroys the product.

KEY NARRATION POINTS:
The global settings establish the defaults, but individual items often need different behavior. On each Item Card, you will find per-item fields that override the global configuration. CASE Size defines the specific packaging for that item. Enable CASE Ledger can be toggled per item — you might track cases for frozen goods but not for bulk spices. Input in Alt UOM or Base UOM determines which unit the system expects users to enter first on documents.
